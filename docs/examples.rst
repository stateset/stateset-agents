Examples
========

Runnable examples live in the repo `examples/` directory.

Common starting points:

- `examples/quick_start.py`: lightweight demo (stub-friendly).
- `examples/finetune_kimi_k25_gspo.py`: Kimi-K2.5 fine-tuning with GSPO.
- :doc:`qwen3_5_starter`: built-docs quick start for post-training ``Qwen/Qwen3.5-0.8B``.
- :doc:`kimi_k2_6_starter`: built-docs quick start for post-training ``moonshotai/Kimi-K2.6``.
- ``docs/QWEN3_FINETUNING_GUIDE.md``: repository walkthrough for the same starter path.
